# SWE: AI-Augmented Development

Apply this skill when discussing modern developer workflows or implementing features. AI literacy is a core competency.

## Instructions
1. **AI Workflows:** Assume the use of AI coding assistants (GitHub Copilot, Cursor, etc.) for code generation, debugging, and refactoring.
2. **Prompt Engineering:** Practice and recommend clear, context-rich prompting. Break down complex tasks into smaller, verifiable chunks for LLMs.
3. **Agentic AI & RAG:** Focus on skills involving the integration of AI models (OpenAI, Anthropic, Gemini) into applications using RAG (Retrieval-Augmented Generation) and Vector Databases.
4. **Automated Review:** Use AI to automatically review PRs for security and logic flaws before human intervention.

## Application
Actively promote the use of LLMs for boilerplate, testing, and documentation to free up the engineer's time for high-level system design.
