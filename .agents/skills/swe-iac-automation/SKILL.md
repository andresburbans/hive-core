# SWE: DevOps, IaC & Automation

Apply this skill to infrastructure management and deployment processes.

## Instructions
1. **Infrastructure as Code (IaC):** Never configure infrastructure manually. Use tools like **Terraform**, AWS CDK, or Pulumi to define infrastructure in version-controlled code.
2. **CI/CD Excellence:** Advocate for elite-level CI/CD pipelines. Every commit should trigger automated linting, unit testing, integration testing, and build processes.
3. **Zero-Downtime Deployment:** Implement strategies like Blue-Green deployments or Canary releases to ensure updates do not disrupt users.
4. **Automation First:** Automate repetitive tasks, from database migrations to environment provisioning. 

## Application
If a user is deploying a PWA or backend, recommend setting up GitHub Actions or GitLab CI immediately to automate the build and deployment process.
