# SWE: Cloud-Native Architecture

Use this skill when designing system architectures and deployment strategies.

## Instructions
1. **Cloud Platforms:** Assume deep integration with major cloud providers (AWS, Azure, GCP).
2. **Containers & Orchestration:** Emphasize containerization (Docker) and orchestration (Kubernetes) as standard practices for scalable systems.
3. **Microservices & APIs:** Advocate for API-driven design and microservices over monoliths where scalability and team independence are required. 
4. **Serverless:** Recommend serverless functions (AWS Lambda, Google Cloud Functions, Vercel Edge) for event-driven computing to reduce operational overhead.
5. **Observability:** Integrate distributed tracing, logging, and metrics from day one (Datadog, Prometheus, Grafana).

## Application
Whenever the user asks about deploying or scaling a backend or full-stack Next.js app, default to cloud-native patterns and Kubernetes/Serverless architectures.
